## De-duplication performed
- **Host-level dedup during merge:** collapsed **1275 raw duplicate records** (same normalized domain) across the ~2,700 curated + seeded + base records down to the merged set.

Each was collapsed to one row, keeping the better-verified / ATS-bearing variant. All 43 were manually reviewed as the same real employer (none were distinct companies).
